# FoodEase: Online Food Delivery Platform

FoodEase is a full-stack web application that enables users to browse restaurant menus, place orders, and track deliveries in real time.

## Features

- Browse food menu items
- User authentication (login/signup)
- Add to cart and place orders
- Payment gateway integration (test mode)
- Real-time order tracking (simulated)
- Responsive UI (React)
- REST API backend (Node.js, Express, MongoDB)

## Running Locally

### Backend

```bash
cd backend
npm install
# Add .env file with MONGO_URI variable
npm start
```

### Frontend

```bash
cd frontend
npm install
npm start
```
